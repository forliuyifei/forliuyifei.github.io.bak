  • ~ ins：插入（下划线）
    ++下划线++ => <ins>下划线</ins> => 下划线

  • ~ sub：下标
    H~2~0 => H<sub>2</sub>O => H20

  • ~ sup：上标
    29^th^ => 29<sup>th</sup> => 29th

  • ~ emoji：emoji 表情
    Shortcuts 写法：:/ :| :) => 😕 😐 😃
    Cheat Sheet 写法：:joy: :thinking: :heart: :cn: => 😂 🤔 ❤️ 🇨🇳

  • ~ mark：高亮
    ==高亮文字== => 高亮文字

  • ~ task-lists：GitHub 风格的 Task List
    - [ ] 未完成 - [x] 已达成
