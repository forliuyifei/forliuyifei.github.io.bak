「记录」本站用到的插件

2020 七月 9日 , 星期四 , 11:33 中午 原创
插件 简介 用法/示例/心得 安装命令（站点根目录右键Git Bash）
hexo-abbrlink 短链生成 自动生成短链例如abbrlink: 18338（据说此种格式有利于SEO优化）。如有必要可直接修改Front-matter（文章信息头）进行自定义abbrlink: 我的文章-my-posts-12138，尽量使用字母和数字，空格用-替代，提高链接可读性。 npm install hexo-abbrlink --save
hexo-admin 可视化编辑 不是很好用，经常busy npm install --save hexo-admin
hexo-generator-index-pin-top 文章置顶 在Front-matter插入top:100，数字越大文章在首页的位置越靠前。 先npm uninstall hexo-generator-index --save 再npm install hexo-generator-index-pin-top --save
hexo-blog-encrypt 文章加密 在文章信息头插入password: mikemessi abstract: 有东西被加密了, 请输入密码查看. message: 您好, 这里需要密码.。解密后部分元素可能无法正常显示或者表现，见详细配置 npm install --save hexo-blog-encrypt
hexo-hide-posts 文章隐藏 默认hidden: true用于隐藏，hidden可自定义为自己喜欢的名字。 npm install hexo-hide-posts --save
hexo-generator-feed 生成feed订阅（RSS） 默认只获取文章摘要或者前140个字符 npm install hexo-generator-feed --save
hexo-generator-sitemap 站点地图 sitemap 将生成的sitemap.xml提交给搜索引擎（百度拉跨，谷歌收录极快，Bing也还行）。搜索site:loafing.cn查看被收录情况。 npm install hexo-generator-sitemap --save
hexo-tag-aplayer 音乐播放 aplayer {% meting "7635339557" "tencent" "playlist" "order:random" "preload:none" %} 建议新建一个歌单专门用于网站播放，歌曲越多加载越慢。 npm install --save hexo-tag-aplayer
hexo-tag-dplayer 视频播放 dplayer {% dplayer height=25% "url=https://lyfclub.herokuapp.com/videos/%E6%8A%B1%E6%8A%B1.mp4" "pic=https://test.jpg" "loop=yes" "theme=#FADFA3" "autoplay=false" "preload=false" %}比video标签好用。功能强大，支持字幕引入。 npm install hexo-tag-dplayer --save
hexo-tag-instagram Instagram分享 见后文 npm install hexo-tag-instagram --save
hexo-filter-nofollow 不跟踪外链 对SEO友好。 可排除自定义域名。 npm i hexo-filter-nofollow --save
